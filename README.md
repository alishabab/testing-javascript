# Testing JavaScript with Jest

> In this project we wrote unit test cases for various string and array methods using Jest testing framework.
> We created a string manipulation and calculator module with various functions as well as an 'arrayAnalyzer' function.
> Multiple tests were created via Jest to drive the development following TDD practices.




## Built With
- JavaScript
- Jest

## Getting Started

To get a local copy up and running follow these steps:

### Prerequisites

- Basic knowledge of JavaScript
- A Code Editor
- Git

### Usage

- Fork/Clone this project to your local machine
- Run `npm install` to download dependencies
- Run `npm test` to run test cases
